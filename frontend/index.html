¿Qué es Plotter 2D?

Plotter 2D es un sistema innovador que combina tecnología avanzada con creatividad. Este proyecto permite a los usuarios dibujar en un canvas digital interactivo a través de una aplicación web y, posteriormente, replica los dibujos en un plotter físico en tiempo real. Diseñado para educación, arte y prototipos tecnológicos, Plotter 2D simplifica la conexión entre el mundo digital y físico mediante herramientas modernas de la nube y protocolos de comunicación eficientes.

¿Cómo funciona?

Plotter 2D utiliza una arquitectura escalable basada en AWS para garantizar eficiencia y seguridad. Los usuarios dibujan en una app web, se autentican mediante Cognito y envían sus dibujos al backend, donde son procesados y transmitidos al plotter físico mediante AWS IoT Core y el protocolo MQTT.

Infraestructura

La infraestructura de Plotter 2D está diseñada para aprovechar los servicios en la nube de AWS y garantizar un flujo de datos eficiente y seguro desde la aplicación web hasta el dispositivo físico. Los usuarios interactúan con la interfaz de usuario alojada en Amazon S3, accesible a través de Amazon CloudFront para una entrega rápida y segura. La autenticación se gestiona mediante Amazon Cognito, garantizando acceso protegido. El API Gateway recibe los datos de dibujo, que luego son procesados por funciones Lambda y enviados al dispositivo a través de AWS IoT Core utilizando el protocolo MQTT.
